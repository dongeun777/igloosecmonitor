User Guide

Solution Process

Step1

1. Get Started 클릭

○ 버튼을 클릭하면 Microsoft Azure Marketplace로 이동

Step2

2. Contact Me 클릭

○ 정보 입력 : Name, Email, Job Title, Company, Country, Phone Number

※ Azure Marketplace 계정 필요

Step3

3. 이글루 시스템 접속 안내 메일 수신

○ Step2에서 입력한 Eamil 계정으로 시스템 접속 안내 메일 수신(최대 5분 소요)

○ URL 접속 후 이글루 시스템 로그인

※ 발급된 Password는 임시용 이므로 로그인 후 변경

Step4

4. 고객 시스템 정보 등록

○ 장비 정보 : 장비 - 일일 로그량(GB) 등록

○ 기간 옵션 : 종량제, 예약제(1년, 3년) 선택

○ 정보 등록 후 VM 배포 시작(약 5분 소요)

※ 배포가 완료되면 자동으로 모니터링 시스템으로 화면 전환(모니터링 시스템 이용 가능)

Step5

5. 솔루션 배포완료 안내 메일 수신

○ Step2에서 입력한 Eamil 계정으로 시스템 접속 안내 메일 수신

○ 배포가 완료되면 TM URL로 접속 및 로그인

※ 발급된 Password는 임시용 이므로 로그인 후 변경

Step6

6. 고객장비 에이전트 설치 및 로그 소스 등록

○ TM 가이드를 참고하여 로그 소스 등록 완료

※ TM 가이드 : 사용자 메뉴 > 프로그램 정보 > 메뉴얼 다운로드
